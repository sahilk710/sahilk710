# Sahil Kasliwal — Portfolio

Personal portfolio of **Sahil Kasliwal**, Data Scientist & AI Engineer.

Live site: [https://sahilk710.github.io/sahilk710/](https://sahilk710.github.io/sahilk710/)

## Tech Stack

- React 18 + TypeScript + Vite
- Three.js / React Three Fiber (3D character & physics-based tech sphere)
- GSAP (ScrollTrigger, ScrollSmoother)
- GitHub Actions for deployment

## Local Development

```bash
npm install
npm run dev
```

## Deployment

Pushes to `main` automatically deploy via GitHub Actions to GitHub Pages.
